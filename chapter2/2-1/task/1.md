# Task1
1. HTML 是什么？

    HyperText Markup Language 超文本标记语言

2. HTML 的基本结构？

包含，html, head, body
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

3. HTML 注释？

```html
<!-- hello -->s
```

4. HTML <head> 中包含哪些标签？

meta, title
